# Keyboard Shortcuts

Moltz supports comprehensive keyboard navigation for power users.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + N` | New conversation |
| `Cmd/Ctrl + K` | Open search dialog |
| `Cmd/Ctrl + ,` | Open settings |
| `Cmd/Ctrl + \` | Toggle sidebar |
| `Cmd/Ctrl + /` | Focus chat input |
| `Cmd/Ctrl + Shift + Space` | Open quick input window |
| `Escape` | Close dialogs/modals, hide sidebar on mobile |

## Chat View

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Enter` | Send message (while typing) |
| `Shift + Enter` | New line (while typing) |
| `Escape` | Stop generating (during streaming) |

## Sidebar Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Navigate between conversations |
| `Arrow Up/Down` | Navigate between conversations |
| `Enter` or `Space` | Select conversation |
| `Delete` or `Backspace` | Delete conversation (with confirmation) |

## Dialog Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Navigate between fields and buttons |
| `Escape` | Close dialog |
| `Enter` | Confirm action (in confirmation dialogs) |
| `Arrow Up/Down` | Navigate search results (in search dialog) |

## Accessibility Features

- **Focus visible indicators**: All interactive elements show a clear focus ring when navigated with keyboard
- **Focus trapping**: Dialogs trap focus within them for easier navigation
- **Skip links**: Screen reader users can skip to main content
- **ARIA labels**: All controls are properly labeled for screen readers
- **Reduced motion**: Respects `prefers-reduced-motion` for animations
- **High contrast**: Adapts to high contrast mode preferences

## Tips

- Press `?` anywhere in the app to see available shortcuts (coming soon)
- All actions can be performed without a mouse
- Focus indicators follow WCAG 2.1 Level AA guidelines
